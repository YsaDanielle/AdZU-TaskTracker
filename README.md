# AdZU Student Task & Deadline Tracker

A clean university-themed web app for Ateneo de Zamboanga University students to organize subjects, track deadlines, and mark work as done. The experience is designed like a light academic dashboard: fast to use, visually polished, and easy to demo to recruiters.

## Features

- Add subjects with custom color coding
- Add tasks with due dates, notes, and priority tags
- Track work across `To Do`, `In Progress`, and `Done`
- Switch between board and calendar views
- Filter tasks by search, status, and priority
- Persist data locally with `localStorage`
- Reset the demo data for portfolio walkthroughs

## Tech Stack

- Flask for lightweight local serving
- React 18 rendered in the browser
- Tailwind CSS for styling
- `localStorage` for persistence

## Run Locally

1. Open a terminal in this folder.
2. Install Flask if needed:

```bash
pip install flask
```

3. Start the app:

```bash
python3 app.py
```

4. Open the local URL shown in the terminal, usually `http://127.0.0.1:5000`.
